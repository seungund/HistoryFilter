# History Filter

## 특정 도메인의 방문 기록 자동 삭제 크롬 확장 프로그램

> 개별적인 최상위 도메인의 방문 기록 자동 삭제
> 개발기간 : 2026.03

---

## History Filter 소개

특정 사이트 방문 시 **자동으로 히스토리를 삭제**하여
웹사이트 방문 기록을 남기지 않도록 하는 **Chrome Extension**

사용자가 지정한 도메인에 대해 방문 기록을 감지하고 즉시 삭제

---

## 주요 기능

- 특정 도메인 방문 시 **자동 히스토리 삭제**
- 확장 프로그램 ON/OFF 스위치 제공
- 차단 도메인 **추가 / 삭제 기능**
- 실시간 반영되는 UI
